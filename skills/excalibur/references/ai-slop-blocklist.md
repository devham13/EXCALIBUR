# Excalibur — AI-slop blocklist

Список для post-write scan (из практик [justinbao19/blog-writing-skill](https://github.com/justinbao19/blog-writing-skill) и Excalibur BLOG anti-haltura).

## Veto phrases (русский) — удалить или переписать

- в современном мире
- давайте разберёмся / давайте поговорим
- подводя итог / в заключение / резюмируя
- стоит отметить / важно понимать / не секрет что
- на сегодняшний день / в наше время
- в этой статье мы рассмотрим / вы узнаете
- безусловно / несомненно / undoubtedly
- играет важную роль / ключевую роль
- комплексный подход / всесторонний
- уникальная возможность / революционный
- на 100% / гарантированно / лучший в мире
- landscape / leverage / delve / tapestry / realm (англ. в русском тексте)

## Structural tells

- Более **2** «шероховатостей-самоправок» на статью
- Три и более абзаца подряд одинаковой длины и ритма
- Каждый H2 начинается с «Что такое…» / «Почему важно…»
- Списки из ровно 5 пунктов с одинаковой грамматикой без содержания
- FAQ вопросы не из queries темы

## Fact tells

- «Исследования показывают» без named source
- Круглые проценты (73%, 85%) без источника в research-notes
- «Многие эксперты считают» без имени/организации

## Scan rule

Excalibur GEO QA: **≤ 1** hit из veto list → переписать; **≥ 2** → `FIX_REQUIRED`.
